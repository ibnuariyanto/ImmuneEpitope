\subsection{Overview}\label{overview}

This report implements a comprehensive MHC Class I epitope analysis
pipeline based on NetMHCpan output. It standardizes inputs, assigns
binding classes, scores promiscuity across HLA alleles, filters
candidates using simple physicochemical features, and exports
publication-ready figures and tables.

\subsection{2. Binding Classification}\label{binding-classification}

NetMHCpan convention (EL \%ile):

\begin{itemize}
\tightlist
\item
  Strong binder (SB): EL \%ile ≤ 0.5
\item
  Weak binder (WB): EL \%ile ≤ 2.0
\item
  Non-binder (NB): otherwise
\end{itemize}

\subsection{3. Promiscuity Scoring}\label{promiscuity-scoring}

Scoring:

\begin{itemize}
\tightlist
\item
  SB = 2 points
\item
  WB = 1 point
\item
  Promiscuous: binds ≥ 3 alleles (SB+WB)
\end{itemize}
