\section{Background}\label{background}

This report presents an in-silico MHC-II binding analysis of peptide
candidates for vaccine epitope selection. Binding affinity was predicted
using \textbf{NetMHCIIpan} (EL method). Lower EL percentile rank
indicates stronger predicted binding: \textbf{\textless= 2\%} = strong
binder, \textbf{\textless= 10\%} = moderate binder.
